# Incident Notes and Postmortems

This directory contains controlled incident simulations and postmortems for the LFCS Study Dashboard.

The purpose is to demonstrate operational response discipline:

- detection
- triage
- mitigation
- recovery
- verification
- documentation
- follow-up actions

## Incidents

| Incident | Type | Status |
|---|---|---|
| INCIDENT-001 — LFCS Dashboard Controlled Outage Drill | Local app outage simulation | Resolved |

## Notes

These are local drills, not claims of real production on-call experience.

They are designed to show practical readiness for observability, DevOps, cloud operations, and SRE-adjacent work.
